\# Variables Y Operadores



\## Respuestas Teóricas



\### ¿Para qué se utilizan los operadores lógicos en programación?



Los operadores lógicos permiten combinar condiciones para tomar decisiones dentro del programa.  

Se utilizan principalmente en estructuras de control como `if` o `while`, permitiendo evaluar múltiples condiciones al mismo tiempo.



\### ¿Por qué es importante declarar correctamente el tipo de dato de una variable?



Porque cada tipo de dato ocupa distinta memoria y se comporta diferente.  

Declarar correctamente evita errores, permite operaciones correctas y mejora el rendimiento del programa.

\## Justificación Técnica



\### Clasificación de edad

Se utilizó `if-else` porque permite evaluar rangos numéricos de forma clara y ordenada.



\### Día de la semana

Se utilizó `switch` porque se evalúa un único valor con múltiples opciones posibles, lo que hace el código más limpio.



\### Verificación de acceso

Se utilizó `if-else` porque se deben evaluar múltiples condiciones dependientes entre sí.
